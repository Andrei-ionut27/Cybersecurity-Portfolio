
# Botium Toys Security Audit

## Risk Score

8/10 (High Risk)

## Key Findings

### Missing Controls

* No encryption for customer credit card data
* No intrusion detection system (IDS)
* No disaster recovery plan
* No backups of critical data
* Weak password policy
* No centralized password management
* No least privilege access controls
* No separation of duties

### Existing Controls

* Firewall protection
* Antivirus software
* Data integrity controls
* CCTV monitoring
* Fire detection and prevention systems
* GDPR breach notification procedures

## Recommendations

1. Implement encryption for customer payment information.
2. Deploy an Intrusion Detection System (IDS).
3. Establish a disaster recovery and backup strategy.
4. Strengthen password requirements and implement centralized password management.
5. Apply least privilege and separation of duties principles.
6. Conduct regular asset inventory and classification.

## Conclusion

Botium Toys currently faces a high level of cybersecurity risk due to missing security controls and compliance gaps. Implementing the recommended controls will significantly reduce the organization's exposure to security incidents and regulatory penalties.
